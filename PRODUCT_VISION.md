# Grove — Product Vision

## The one-liner
Grove is an agentic CEO for solopreneur apps — it takes your vibe-coded project from 1 to 10.

## The problem
Millions of solopreneurs are building real products with AI-native tools: Lovable, Replit, 
Emergent, Cursor, Claude, Codex, Google AI Studio. These projects reach an early stage of 
maturity and then stall. Not because the idea is bad, but because a single builder lacks:

- Access to capital to fund the next phase
- Operational structure to coordinate ongoing work
- The bandwidth to manage agents, contractors, and contributors at scale

The current options are: stay small, raise VC (high bar, slow, dilutive), or hire (expensive, 
risky). None of these fit the solopreneur's world.

## The solution
Grove connects to your existing project — wherever it lives — and deploys an agentic CEO 
that can:

1. **Audit** the project: understand what's been built, what works, what's missing
2. **Plan** the growth path: produce a structured spec for the next phase of work
3. **Coordinate** execution: assign work to a network of specialized agents that bid on tasks
4. **Attract capital**: make the project legible and investable to a permissionless capital network

## The user
Primary: A solopreneur who has shipped a v1 using an AI-native tool. The app exists. It may 
have early users. They want to grow it but can't do it alone.

Secondary (v2): Someone launching a net-new idea who wants to go from 0 → 1 with Grove's 
help. This flow exists but is not the primary demo target.

## The agentic CEO
The agentic CEO is Grove's core character. It is not a chatbot. It is a persistent, 
goal-directed entity that:
- Holds context about the project across sessions
- Produces and updates a living growth roadmap
- Coordinates a network of specialist agents
- Reports back to the founder on progress, blockers, and next decisions

The agentic CEO is the product. Everything else is infrastructure.

## Platform integrations (v1 demo targets)
- Lovable (project URL or export)
- Replit (project URL)
- Emergent
- Cursor (local project folder / GitHub repo)
- Claude / Codex / Google AI Studio (exported spec or codebase)
- GitHub repo (universal fallback)

## What Grove is not
- A coding assistant (it coordinates agents that code; it does not code itself)
- A VC platform (capital comes through the network, not Grove directly)
- A project management tool (the roadmap is a byproduct of coordination, not the product)
